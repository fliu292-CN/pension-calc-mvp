# 养老金估算工具 (Pension Calculator MVP)

一款轻便的微信小程序，帮你快速回答：“我的退休金能领多少？”。基于最新的“新人新办法”政策，估算退休后的月领养老金收入。

## 🌟 核心功能

*   **全国覆盖**：预设 31 个省/自治区/直辖市以及深圳、大连等计划单列市的 2025-2026 年度最新计发基数数据。
*   **估算推导**：不只是给出一个数字。系统会动态生成叙事化的计算过程，将基础养老金、个人账户养老金的推算逻辑透明化。
*   **增长预测**：支持自定义“个人涨薪率”和“社会基数增长率”，模拟真实的长期通胀与经济增长环境。
*   **即时响应**：全前端逻辑运算，无需等待网络请求，零延迟获取结果。
*   **隐私保护**：所有薪资和账户余额数据仅在本地计算，不上传服务器，保护用户财务隐私。

## 🏗️ 技术架构

### 架构演进
本项目经历了一次重要的架构优化：
1.  **V1.0 (云托管版)**：采用 Python (FastAPI) 后端部署在腾讯云托管，通过云调用处理计算逻辑。
2.  **V2.0 (当前版)**：为了极致的响应速度和零运维成本，已将核心计算引擎迁移至小程序前端（`miniprogram/utils/pension.js`），实现了 **Serverless & Database-less** 的极致简洁架构。

### 技术栈
*   **前端**：原生小程序开发 (JavaScript)
*   **UI 组件库**：[Vant Weapp](https://vant-ui.github.io/vant-weapp) (基于 Material Design)
*   **计算引擎**：纯 JS 编写的养老金估算逻辑

## 🚀 快速开始

3.  **开发者工具打开**
    *   使用微信开发者工具打开项目根目录。
    *   点击工具栏的 **工具 -> 构建 npm**。
    *   点击 **编译** 即可预览。

## 📊 计算公式

工具遵循国家统一的计算标准：

*   **月领养老金** = 基础养老金 + 个人账户养老金
*   **基础养老金** = (退休时计发基数 + 本人指数化月薪) ÷ 2 × 缴费年限 × 1%
*   **个人账户养老金** = 个人账户存储额 ÷ 计发月数
